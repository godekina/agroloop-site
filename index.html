AgroLoop: Circular Farming Innovation

Location: Kaduna, Nigeria

AgroLoop is a working circular farming system that converts organic waste into high-protein maggot feed for sheep, goats, and pigs. We use precision tools and aim to scale this model using cloud technologies like AWS.

What We Do:

  • Convert market, farm & brewery waste into maggot protein
  • Use maggots to produce livestock feed
  • Feed goats, sheep, and pigs with formulated feed
  • Recycle animal waste back into maggot production
  • Apply frass as organic fertilizer
